⚠️ 如何安装英语语音包
Windows 系统：
1. 按 Win + I 打开 Windows 设置
2. 进入 时间和语言 → 语音
3. 点击 "添加语音"
4. 选择 英语（美国） 或 英语（英国）
5. 等待下载完成后 重启应用

Mac 系统：
1. 打开 系统偏好设置 → 辅助功能 → 语音内容
2. 点击 "系统语音" 下拉菜单
3. 选择 "自定" 并下载英语语音
